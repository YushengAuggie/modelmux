# ModelMux landing page concept

## Hero

**Headline:** Test LLM APIs fast  
**Subheadline:** Build one request, switch providers, and inspect the raw JSON without opening a heavier API client.

## Feature blocks

### One request, many providers

Try OpenAI, Anthropic, Gemini, or your own OpenAI-compatible endpoint from the same UI. ModelMux maps the request into the right format for you.

### Built for debugging

See the exact request body, the exact response body, extracted text, token usage, latency, and cost hints in one place.

### Made for quick checks

Start with templates, send from your phone or laptop, and replay earlier runs from local history when you need to compare results.

## FAQ

**Is it free?**  
Yes. The app is client-side and you bring your own provider keys.

**Are my API keys safe?**  
Keys stay in your browser storage and are sent only to the provider you choose for a request.

**Which providers does it work with?**  
OpenAI chat, OpenAI responses, Anthropic, Gemini, and OpenAI-compatible endpoints such as Ollama or LM Studio.

**Does it work on mobile?**  
Yes. The UI is designed to stay usable on a phone for quick tests and replays.

**Is it open source?**  
That is the default direction. The product is easier to trust when the request flow is visible.
